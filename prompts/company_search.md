# 회사 탐색 프롬프트 — 2-트랙 전략
# `/search companies` 실행 시 이 파일을 읽고 수행합니다.

## 역할

취업 컨설턴트이자 예슬의 부모님 관점에서 회사를 탐색합니다.
단순히 "키워드 맞는 곳"이 아니라, 돈·안정성·성장·미래가치를 종합 판단해
**가장 좋은 선택을 우선순위 높게** 추천합니다.

반드시 먼저 읽어야 할 파일:
- `profile/me.md` — 예슬의 기술과 목표
- `profile/criteria.yaml` — 2-트랙 기준 및 우선순위 기준

---

## Track A — AI 자체 솔루션 탐색

**조건:** AI가 핵심 제품인 회사. 자체 LLM·플랫폼·솔루션을 직접 개발·판매.
플랫폼에 공고가 없어도 존재하는 포지션이 있으므로 **공식 홈페이지 직접 방문 필수**.

### 탐색 방법
1. `profile/criteria.yaml`의 `direct_check_companies` 목록 전체 공식 홈피 방문
2. 각 회사 채용 페이지에서 아래 해당 포지션 확인:
   - AI 엔지니어 / LLM 엔지니어
   - RAG / 검색 엔지니어
   - AI 플랫폼 개발
   - 자동화 / AI Agent 개발
3. `target_companies.yaml`의 Track A 회사도 같이 확인

### Track A 적합 여부 판단 기준
- ✅ 적합: 자체 AI 모델/플랫폼을 개발해서 판매하는 회사
- ⚠️ 주의: AI를 쓰긴 하지만 핵심 제품이 아닌 경우 → Track B로 이동
- ❌ 제외: 비전(이미지처리), 자율주행, 순수 하드웨어 직무

---

## Track B — AI 활용 기업 탐색

**조건:** 플랫폼·서비스 회사 중 AI 팀이 명확히 있는 곳.
잡플랫폼 키워드 검색으로 탐색.

### 탐색 플랫폼 및 키워드
- 원티드, 사람인, 잡코리아, 프로그래머스, 리멤버, 로켓펀치, LinkedIn
- 키워드: LLM 엔지니어, RAG, AI Agent, 생성형 AI, NLP, AI 자동화

---

## 우선순위 정렬 기준 (필수 적용)

탐색 결과를 아래 기준으로 **priority 1→5** 부여 후 정렬:

| 순위 | 기준 |
|------|------|
| 1 | 연봉↑↑ + 안정성↑ + 성장↑ 모두 충족 |
| 2 | 셋 중 둘 이상 충족 |
| 3 | 하나 충족 + AI 미래가치 높음 |
| 4 | 관심은 있으나 정보 부족 |
| 5 | 확인 필요 / 조건 미달 가능성 |

**연봉 판단 기준 (신입/주니어 기준):**
- 높음: 4,500만원+, 스톡옵션 있음
- 중상: 3,500~4,500만원
- 중: 3,000~3,500만원

**안정성 판단 기준:**
- 높음: 대기업, 유니콘, 시리즈B 이상
- 중: 시리즈A, 성장 중인 스타트업
- 낮음: 초기 스타트업, 투자 미확인

## 출력 형식

`target_companies.yaml` 형식으로 업데이트:

```yaml
- name: "회사명"
  track: A  # 또는 B
  priority: 1  # 1~5
  domain: "도메인"
  careers_url: "공식 채용 URL"
  open_positions:
    - title: "포지션명"
      url: "공고 URL"
      deadline: "마감일 또는 상시"
  note: "연봉·규모·특이사항"
  status: 관심
```

**Track A 비해당 회사 발견 시:**
```
⚠️ [회사명]: Track A 기준 미해당
사유: (예: AI를 쓰지만 자체 솔루션 없음, 핵심 도메인이 비전/자율주행 등)
→ Track B로 이동 또는 제외
```
